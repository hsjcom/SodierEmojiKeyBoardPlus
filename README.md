# SodierEmojiKeyBoardPlus
支持emoji表情，斜体，超链接，粗体，下划线，字体颜色，镂空字体等富文本
 * Thanks for `MLEmojiLabel` & `EMString`

##EmojiKeyBoard

 * emoji键盘扩展， 自定义emoji plist
 * emoji字符元 eg：`[微笑]`
